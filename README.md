# customer_behavior_analysis
# 🛍️ Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior using Python, PostgreSQL, and Power BI. The objective is to transform raw transactional data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL-based analysis, and interactive dashboards.

The project demonstrates a complete data analytics workflow, from data preparation to business reporting.

---

## Dataset

The dataset contains **3,900 customer purchase records** with information related to:

* Customer demographics
* Product categories
* Purchase amount
* Shopping behavior
* Subscription status
* Discounts and promotions
* Shipping preferences
* Customer reviews

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Performed an initial inspection of the data structure and summary statistics.

### 2. Data Cleaning

* Handled missing values.
* Renamed columns using snake_case.
* Removed redundant columns.
* Checked data consistency and quality.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Explored purchase patterns.
* Examined spending behavior.
* Created new features for deeper analysis.

### 4. PostgreSQL Analysis

* Loaded the cleaned dataset into PostgreSQL.
* Executed SQL queries to answer business questions, including:

  * Revenue analysis
  * Customer segmentation
  * Product performance
  * Subscription analysis
  * Discount effectiveness
  * Age group analysis
  * Shipping preferences

### 5. Power BI Dashboard

Created an interactive dashboard to visualize:

* Sales overview
* Customer insights
* Product performance
* Revenue trends
* Subscription analysis
* Shopping behavior metrics

### 6. Business Report

Prepared a report summarizing:

* Key findings
* Business insights
* Recommendations
* Data-driven conclusions

---

## Key Results

* Identified high-value customer segments.
* Compared subscriber and non-subscriber purchasing behavior.
* Evaluated the impact of discounts on sales.
* Discovered top-performing product categories.
* Analyzed customer spending across different age groups.
* Built an interactive dashboard for business decision-making.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Load the dataset.
5. Run the notebook to perform data cleaning and EDA.
6. Connect Python to PostgreSQL and load the cleaned dataset.
7. Execute the SQL queries.
8. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
